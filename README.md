# Autonomous Agents – Lab Projects (2025)

This repository contains laboratory projects for the course Autonomous Agents at Technical University of Crete.

## Contents
- **Lab 01 – Webots & Nao controller**
  - Introduction to Webots
  - Implementation of a simple Nao controller to score goals
  - Simple static enviroment

- **Lab 02 – Webots & e-puck Navigation**
  - Braitenberg-based controller for the e-puck robot  
  - Left/right wall-following behavior (forward & reverse)  
  - Obstacle avoidance in a randomized maze (Rat’s Life scenario)

- **Lab 03 – Gazebo Robot Modeling & Simulation**
  - Custom differential-drive robot model (SDF)  
  - World creation and environment setup  
  - Differential motion and sensor integration (IMU, contact, LiDAR)

- **Lab 04 – ROS 2 + Gazebo Integration**
  - URDF-based robot description  
  - ROS 2 ↔ Gazebo bridge  
  - RViz visualization  
  - Turtlebot and rrbot experiments  
  - Launch files and ROS node control

## Notes

- Each lab folder includes:
